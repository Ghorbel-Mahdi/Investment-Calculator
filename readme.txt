# React Investment Calculator

This is a simple investment calculator built using React. It calculates the potential returns on different investment options.

## Installation

1. Clone this repository.
2. Navigate to the project directory in your terminal.
3. Run `npm install` to install the project dependencies.
4. Run `npm start` to start the development server.

## Dependencies

- "@testing-library/jest-dom": "^5.16.5"
- "@testing-library/react": "^13.4.0"
- "@testing-library/user-event": "^13.5.0"
- "react": "^18.2.0"
- "react-dom": "^18.2.0"
- "react-scripts": "5.0.1"
- "web-vitals": "^2.1.4"

## Available Scripts

- `npm start`: Starts the development server.
- `npm build`: Builds the production-ready version of the app.
- `npm test`: Runs tests.
- `npm eject`: Ejects the app from `react-scripts` configuration.
